# alertapp
Log Alert Application in Vuejs


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

# node-api
Log Alert api endpoints


## Project setup
```
npm install express

```

### Compiles and run the server
```
node server.js
```
